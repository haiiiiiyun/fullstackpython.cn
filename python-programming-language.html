Python 编程语言

Python 编程语言是一个 开源的, 广泛应用于 开发软件应用的工具。

Python 能做什么？

Python 通常用于 构建 并 部署 Web 应用 和 Web API。 Python 也可以用于分析及可视化 数据 和 软件测试，即便待测试的软件不是用 Python 写的也可以。
